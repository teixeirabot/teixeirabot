OpenClaw + TeixeirixBot

Resumo rápido do que é o OpenClaw e como este assistente foi configurado.

O que é o OpenClaw

  • OpenClaw é uma plataforma para correr assistentes de IA com acesso a ferramentas reais (ficheiros, shell, browser, mensagens, etc.).
  • Permite automação prática: editar código, correr comandos, pesquisar, controlar sessões e integrar com canais como Telegram.
  • Funciona com memória em ficheiros no workspace, para manter contexto entre sessões.
